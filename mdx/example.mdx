> This document demonstrates how to use the advanced code highlighting features powered by `rehype-pretty-code`.

### Highlighting Specific Lines

To highlight one or more lines, add a range in curly braces `{}` after the language identifier. This is useful for drawing attention to specific parts of a code block.

The syntax is ` ```ts{2, 5-7} ` which highlights line 2 and lines 5 through 7.

```tsx {2, 6-8}
import { useState, useEffect } from 'react';

function useUserData(userId) {
  const [user, setUser] = useState(null);

  useEffect(() => {
    fetch(`/api/users/${userId}`)
      .then(res => res.json())
      .then(data => setUser(data));
  }, [userId]);

  return user;
}
```

### Highlighting Specific Words (Tokens)

To highlight specific words or tokens, enclose them in forward slashes `/` within the curly braces. This is great for emphasizing keywords, variables, or function names.

The syntax is ` ```ts{/useState/ /setUser/} `.

```tsx {/useState/ /setUser/}
import { useState, useEffect } from 'react';

function useUserData(userId) {
  const [user, setUser] = useState(null);

  useEffect(() => {
    fetch(`/api/users/${userId}`)
      .then(res => res.json())
      .then(data => setUser(data));
  }, [userId]);

  return user;
}
```

### Combining Highlights & Handling Empty Lines

You can combine line and word highlighting. This example also includes an empty line (line 4) to demonstrate how our `onVisitLine` plugin correctly preserves it for visual spacing, preventing the line from collapsing.

The syntax is ` ```ts{2, 6} /const/ /return/ `.

```ts {2, 6} /const/ /return/
function calculateTotal(price, quantity) {
  const total = price * quantity;

  return total;
}
```

### Adding a Filename

You can also add a `filename` attribute, which `rehype-pretty-code` will render as a title for the code block. This can be combined with any of the highlighting features.

```ts filename="api/steel-client.ts" {3} /Steel/
import Steel from "steel-sdk";

const client = new Steel({ steelAPIKey: "YOUR_STEEL_API_KEY" });

async function main() {
  const session = await client.sessions.create();
  console.log(`View live session at: ${session.sessionViewerUrl}`);
}
```
